# U-Know Frontend Agent Reference

## 📝 1. 연동 기획 명세
- 너잘알(U-Know) 프론트엔드 라우팅 (`/u-know` 하위 경로).
- 퀴즈 생성, 링크 공유, 정답 제출 및 결과 차트 시각화.

## 🤖 2. 목적 및 개발 지침
### 🎯 목적
- 관계형 퀴즈 UX 및 차트 컴포넌트 개발.
### 🛠️ 개발 지침
- 퀴즈 풀이 화면 인터랙션 최적화.

## 🚨 3. 제약 조건
- **[300줄 분리 규칙]:** 단일 파일 코드는 300줄 초과 금지.
- **[Client Component 최소화]:** 서버 컴포넌트 우선 사용.
