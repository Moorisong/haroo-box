# HTSM Frontend Agent Reference

## 📝 1. 연동 기획 명세
- 자아탐험(HTSM) 프론트엔드 라우팅 (`/htsm` 하위 경로).
- 키워드 선택, 테스트 공유, 조하리의 창 결과 시각화.

## 🤖 2. 목적 및 개발 지침
### 🎯 목적
- 자아탐험 UX/UI 컴포넌트 개발.
### 🛠️ 개발 지침
- 폼 입력 시 클라이언트 단 유효성 검증 적용.

## 🚨 3. 제약 조건
- **[300줄 분리 규칙]:** 단일 파일 코드는 300줄 초과 금지.
- **[Client Component 최소화]:** 서버 컴포넌트 우선 사용.
