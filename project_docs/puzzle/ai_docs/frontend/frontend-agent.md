# Puzzle Frontend Agent Reference

## 📝 1. 연동 기획 명세
- 하루퍼즐(Puzzle) 프론트엔드 라우팅 (`/puzzle` 하위 경로).
- Canvas API 및 3D 물리 효과 적용된 퍼즐 드래그 앤 드롭 조작 화면.

## 🤖 2. 목적 및 개발 지침
### 🎯 목적
- `puzzleStore`, `rankingStore` Zustand 상태 관리 연동 및 퍼즐 UI 위젯 개발.
### 🛠️ 개발 지침
- 조각 맞춤 이벤트 최적화 및 렌더링 퍼포먼스 향상.

## 🚨 3. 제약 조건
- **[300줄 분리 규칙]:** 단일 파일 코드는 300줄 초과 금지.
- **[성능]:** 불필요한 리렌더링 방지.
