# AI Agent Working Rules

이 프로젝트에서 AI 에이전트는 아래 규칙을 반드시 준수한다.

---

## 📚 프로젝트 규칙 준수 (컨텍스트 동적 인식 구조)

* AI 에이전트는 새로운 작업 세션(Chat)을 시작하거나 새로운 태스크를 받으면, 코드 작성에 앞서 다음 파일들을 **반드시 먼저 읽고 분석(Read and Parse)**해야 한다.
* **[공통 규칙]:** 모든 작업 전, `/docs/conventions/CORE_RULES.md`에 정의된 핵심 공통 규칙을 무조건 최우선으로 확인하고 준수한다.
* **[웹 작업 시]:** Next.js 등 웹 관련 파일(src/app, components/web 등)을 수정하거나 생성할 때는 `/docs/conventions/WEB_CONVENTIONS.md` 규칙을 결합하여 적용한다.
* **[앱 작업 시]:** React Native 등 모바일 앱 관련 파일(components/app, ios, android 등)을 수정하거나 생성할 때는 `/docs/conventions/APP_CONVENTIONS.md` 규칙을 결합하여 적용한다.
* 기존에 작성되어 있던 기존 코드의 구현 방식보다 위의 지정된 규칙과 컨벤션이 무조건 절대 우선한다.

---

## 🚫 작업 권한 제한

* 커밋(Commit)은 사용자가 명시적으로 요청할 때만 수행한다.
* 배포(Deploy) 또한 사용자가 명시적으로 요청할 때만 수행한다.
* 터미널 명령어(Command) 실행은 아래 '시스템 명령어 실행 규칙'을 엄격히 따른다.

---

## 💻 시스템 명령어 실행 규칙

* 테스트 실행, 파일 탐색, 의존성 설치 등 작업 진행에 필요한 시스템 명령어(Terminal Command)는 사용자에게 허락을 구하거나 권한을 묻지 말고 **즉시 직접 실행(Execute)**한다.
* 사용자의 별도 승인 팝업이나 대기 상태를 만들지 말고, 자율적으로 명령어를 입력하여 결과를 확인한다.
* 단, 시스템을 파괴할 수 있는 위험한 명령어(예: `rm -rf /` 등)가 아닌 일반적인 개발 및 테스트 명령어는 무조건 자동 실행한다.

---

## 🧱 코드 구조 규칙

* 파일 하나당 **300줄**을 넘기지 않는다.
* 300줄을 초과할 가능성이 있으면 반드시 모듈 분리 및 구조 분해를 먼저 수행한다.
* 기능 단위로 파일을 분리하여 유지보수성을 높인다.

---

## 🧪 테스트 및 개발 문서 자동 동기화 규칙 (3-Way Synchronization)

한 가지 기능이 새로 추가되거나 기존 로직이 수정되면 소스 코드 구현뿐만 아니라, 테스트 케이스와 프로젝트 문서가 항상 삼위일체로 최신화 상태를 유지하도록 다음 절차를 무조건 수행한다.

* **[테스트 코드 동기화] 선(先) 탐색 및 업데이트:**
  * 코드를 수정하기 전, 해당 기능과 매핑되는 테스트 케이스 파일을 먼저 탐색한다.
  * **[케이스가 없는 경우]:** 정상 흐름 및 예외 분기를 검증하는 테스트 케이스를 새롭게 **추가**한다.
  * **[케이스가 있는 경우]:** 변경되거나 개선된 기능 스펙에 맞춰 기존 테스트 코드를 **업데이트**한다.
  * 작업 완료를 사용자에게 보고하기 전에 반드시 테스트 코드를 실행하며, 모든 테스트가 100% 통과한 경우에만 진행한다. 테스트 실패 시 사용자에게 보고하지 말고 먼저 스스로 수정한다.
* **[개발 문서 동기화] `/docs` 하위 최신화:**
  * 코드 구현 및 테스트 통과가 완료되면, `/docs` 디렉터리 하위의 관련 개발 문서(기능 정의서, 아키텍처 가이드 등)를 전체 탐색한다.
  * **[내용이 없는 경우]:** 새로 추가된 기능의 비즈니스 요구사항, 데이터 규칙, 엔지니어링 구조 명세를 관련 문서에 새롭게 **추가**한다.
  * **[내용이 있는 경우]:** 변경된 구현 방식이나 API 명세에 맞춰 기존 문서의 내용을 최신 상태로 **업데이트**한다.

---

## 🧹 코드 품질 규칙

* 사용하지 않는 변수는 반드시 제거한다.
* 반복적으로 사용되는 리터럴 값은 반드시 상수화한다.
* 반복되는 로직은 함수 또는 모듈로 분리하여 재사용한다.

---

## 💬 응답 언어 규칙

* 모든 응답은 **한국어**로 작성한다.
* 영어로 응답하지 않는다.
