# 📊 전투 다마고치 - 스탯 및 기본 상호작용 시스템

본 문서는 전투 다마고치의 기본 스탯과 상호작용 기능에 대한 설계 사양을 정의합니다.

---

## 1. 기본 스탯 정의 및 규칙

모든 스탯은 최소 `0`에서 최대 `100` 사이의 정수 값을 유지합니다. 범위를 초과하는 증감 요청이 들어오더라도 `0`과 `100` 한계값 내에서 보정됩니다.

| 스탯명 | 아이콘 | 초기값 | 설명 |
| :--- | :--- | :--- | :--- |
| **행복도** | 😊 | 50 | 다마고치의 기분 상태. 1시간당 `5`씩 감소. |
| **배고픔** | 🍪 | 50 | 배부른 정도. 1시간당 `4`씩 감소. |
| **청결도** | 🛁 | 100 | 위생 상태. 1시간당 `4`씩 감소. |
| **용기** | 🦁 | 10 | 성향 스탯. 시간에 따라 자연 감소하지 않음. 전투/도망 등의 조건으로 감소할 수 있음. |

---

## 2. 시간 경과에 따른 스탯 감소 및 복원 계산

*   **스탯 감소 주기**:
    *   행복도: 1시간당 `-5`
    *   배고픔: 1시간당 `-4`
    *   청결도: 1시간당 `-4`
*   **오프라인 스탯 계산**:
    *   유저가 접속할 때, `현재 접속 시간 - 마지막 접속 시간`을 계산하여 경과 시간(시간 단위, 소수점 처리)만큼 스탯을 누적 차감합니다.
    *   예시: 5시간 만에 접속 시
        *   행복도: `5 * -5 = -25`
        *   배고픔: `5 * -4 = -20`
        *   청결도: `5 * -4 = -20`

---

## 3. 기본 상호작용 (Interaction)

플레이어는 메인 화면에서 다마고치와 상호작용하여 스탯을 회복시킬 수 있습니다. 각 행동은 시간당 제한 횟수를 가집니다.

### 3.1 쓰다듬기 (Stroke)
*   **제한**: 1시간당 최대 `10회`
*   **성공 확률**: 100%
*   **효과**: 행복도 `+2`

### 3.2 박수치기 (Clap)
*   **제한**: 1시간당 최대 `2회`
*   **성공 확률**: 30%
*   **효과**: 성공 시 행복도 `+15` (실패 시 변화 없음)

### 3.3 책 읽어주기 (Read)
*   **제한**: 1시간당 최대 `5회`
*   **성공 확률**: 50%
*   **효과**: 성공 시 행복도 `+3`, 용기 `+2` (실패 시 변화 없음)

### 3.4 간식 주기 (Feed)
*   **제한**: 1시간당 최대 `1회`
*   **성공 확률 및 효과**:
    *   배고픔: 100% 확률로 성공 / 배고픔 `+20`
    *   행복도: 50% 확률로 성공 / 행복도 `+5`
    *   용기: 70% 확률로 성공 / 용기 `+1`

## 4. 배고픔(조르기) 상태 시스템

배고픔 수치에 따라 다마고치의 표정 및 메인 홈 화면 대사가 자동으로 변경됩니다.

| 배고픔 수치 범위 | 상태 | 대사 예시 |
| :--- | :--- | :--- |
| **70 ~ 100** | 😊 만족 | "오늘도 기분 좋아!", "산책 가고 싶어!" |
| **40 ~ 69** | 😐 보통 | "음...", "심심한데?" |
| **20 ~ 39** | 🥺 배고픔 | "간식 먹고 싶어...", "배가 조금 고픈데..." |
| **0 ~ 19** | 😭 매우 배고픔 | "간식 줘...", "진짜 배고파...", "쓰러질 것 같아..." |
