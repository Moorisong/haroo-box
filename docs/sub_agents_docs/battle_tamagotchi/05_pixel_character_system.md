# 🎨 전투 다마고치 - 픽셀 캐릭터 및 외형 조합 시스템

본 문서는 전투 다마고치의 그래픽 사양, 종족 분류, 🌈 색상 변이 및 레이어 조합 시스템의 설계 사양을 정의합니다.

---

## 1. 그래픽 및 캐릭터 사양

*   **그래픽 스타일**: 복고풍의 아기자기한 2D 픽셀 아트.
*   **해상도 규격**: `48x48` 픽셀 아트 기준.
*   **애니메이션**: 걷기, 울기, 졸기, 도망가기, 대기 애니메이션 표현.
*   **오리지널리티**: 기존 외부 IP를 일절 참고하지 않는 순수 창작 캐릭터 리소스 사용.

---

## 2. 알 부화 및 종족

플레이어는 최초 시작 시 알을 선택하고 다마고치 이름을 정해야 합니다. (이름은 중복 허용되지 않음)
알 부화 시 아래 4가지 종족 중 하나가 무작위(또는 선택)로 태어납니다.

*   **🟢 귀염상**: 둥글고 귀여운 스타일의 외형.
*   **🔵 기괴상**: 독특하고 개성 넘치거나 기괴한 실루엣.
*   **🩷 평범상**: 가장 일반적이고 안정적인 외형.
*   **🟡 개성상**: 독특한 비율과 병맛 감성의 실루엣.

---

## 3. 레이어 조합 시스템 (Composite Layers)

다마고치의 캐릭터 이미지는 여러 겹의 레이어를 겹쳐서 실시간으로 합성 및 렌더링합니다.

$$\text{캐릭터 이미지} = \text{본체(색상 반영)} \to \text{표정} \to \text{꽃} \to \text{모자}$$

1.  **본체 (Body)**: 종족별로 5종씩 제공되며, 총 `20종` 존재.
2.  **색상 팔레트 (Color Palette)**:
    *   같은 본체라도 랜덤한 색상 팔레트(`5종`)가 부여되어 색상 변이를 만듭니다.
    *   예시: 연두, 하늘, 연보라, 분홍, 노랑, 회색 등.
3.  **표정 (Expression)**: 현재 스탯 상태에 따라 자동으로 갱신되는 `5종` 표정.
    *   행복, 슬픔, 화남, 졸림, 간식 조르기
4.  **꽃 액세서리 (Flower)**: 산책으로 획득하여 머리 위에 장착하는 액세서리 (`6종`).
    *   민들레, 초록 새싹, 장미, 해바라기, 네잎클로버, 황금꽃
5.  **모자 액세서리 (Hat)**: 탄생 시 무작위로 1개를 가지고 태어나며, 죽을 때까지 고정되는 액세서리 (`10종`).
    *   왕관, 리본, 선글라스, 전투 헬멧, 캡 모자, 종이상자, 냄비, 개구리 모자, 똥 모자, 뚫어뻥

---

## 4. 리소스 조합 및 조합 수

*   **리소스 구성**:

| 분류 | 개수 |
| :--- | :--- |
| **본체** | 20종 |
| **색상 팔레트** | 5종 |
| **표정** | 5종 |
| **꽃** | 6종 |
| **모자** | 10종 |

*   **이론적 조합 수**:
    $$20 \text{ (본체)} \times 5 \text{ (색상)} \times 5 \text{ (표정)} \times 6 \text{ (꽃)} \times 10 \text{ (모자)} = 30,000\text{가지 조합}$$
